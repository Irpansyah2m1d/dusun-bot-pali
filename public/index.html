Pertanyaan Sering Muncul

Apa itu Dusun Bot PALI?

Dusun Bot PALI adalah sebuah project Open Source API yang dikembangkan untuk melestarikan dialek daerah Penukal Abab Lematang Ilir melalui teknologi Kecerdasan Buatan.

Bagaimana cara kerja API ini?

API ini menggunakan metode RAG (Retrieval-Augmented Generation) di mana sistem akan mencari kosa kata yang relevan dari database kamus lokal sebelum diproses oleh model AI seperti Gemini atau Llama.

Apakah saya perlu membayar untuk menggunakan API ini?

Project Dusun Bot sendiri bersifat gratis dan Open Source. Namun, untuk fitur Chat dan Terjemahan, Anda memerlukan API Key dari provider (seperti Google Gemini) yang mungkin memiliki batasan kuota gratisnya sendiri.

Provider AI mana yang paling bagus untuk Bahasa Dusun?

Kami sangat merekomendasikan Google Gemini karena kemampuannya dalam memahami konteks bahasa daerah yang lebih natural dibandingkan model lainnya saat ini.

Bagaimana cara mendapatkan API Key?

Anda bisa mendapatkan API Key Gemini secara gratis melalui Google AI Studio, atau Groq API Key melalui halaman Groq Cloud Console.

Bisakah saya berkontribusi menambah kosa kata?

Sangat bisa! Kami mengundang Anda untuk berkontribusi baik melalui Pull Request di GitHub maupun dengan mengirimkan data kosa kata melalui WhatsApp atau Email kami.

Apakah data kamus akan terus diperbarui?

Ya, kami terus berusaha untuk memperkaya dataset kamus agar hasil terjemahan dan percakapan chatbot menjadi semakin akurat dan luwes.
