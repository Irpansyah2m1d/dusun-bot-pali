Pertanyaan Umum (FAQ)

Apa itu Dusun Bot PALI?

Dusun Bot PALI adalah project API Open Source yang dirancang untuk melestarikan dan mendigitalisasi dialek daerah PALI (Sumatera Selatan) menggunakan bantuan AI.

Bagaimana Cara Menggunakannya secara Gratis?

Dengo cuma butuh API Key dari Google AI Studio atau Groq Cloud. Kedua layanan tersebut memiliki kuota gratis yang sangat cukup untuk penggunaan pribadi atau project kecil.

Bisakah Saya Berkontribusi Menambah Kata?

Sangat bisa! Project ini adalah milik komunitas. Dengo bisa mengirimkan kosa kata baru melalui WhatsApp Pengembang atau langsung melakukan Pull Request di GitHub.
