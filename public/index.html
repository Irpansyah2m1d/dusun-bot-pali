Pertanyaan Umum (FAQ)

Apa itu Dusun Bot PALI?

Dusun Bot PALI adalah project API Open Source yang dirancang untuk melestarikan dan mendigitalisasi dialek daerah PALI (Penukal Abab Lematang Ilir, Sumatera Selatan) menggunakan bantuan AI. Dilengkapi kamus 2800+ kosa kata, chatbot AI, dan penerjemah kalimat.

Bagaimana Cara Menggunakannya secara Gratis?

Dengo cuma butuh API Key dari Google AI Studio atau Groq Cloud. Kedua layanan tersebut memiliki kuota gratis yang sangat cukup untuk penggunaan pribadi atau project kecil.

Bisakah Saya Berkontribusi Menambah Kata?

Sangat bisa! Project ini adalah milik komunitas. Dengo bisa mengirimkan kosa kata baru melalui WhatsApp Pengembang atau langsung melakukan Pull Request di GitHub.

Bagaimana Keamanan API Key Saya?

API Key Anda dikirimkan langsung dari client ke endpoint serverless kami dan hanya digunakan untuk autentikasi ke provider AI (Gemini/Groq). Kami tidak menyimpan API Key Anda di database kami.

Di mana Saya Bisa Menemukan Dokumentasi Lengkap?

Dokumentasi teknis lengkap beserta contoh script dalam berbagai bahasa pemrograman dapat Anda temukan di repositori GitHub resmi kami.
